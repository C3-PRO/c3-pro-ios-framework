  • transaction(_:block:)

    Runs a transaction with the given mode.

    Note

    Transactions cannot be nested. To nest transactions, see savepoint(), instead.

  • Parameters:

    • mode: The mode in which a transaction acquires a lock.

    Default: .Deferred - block: A closure to run SQL statements within the transaction. The transaction will be committed when the block returns. The block must throw to roll the transaction back.
